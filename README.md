# emo

Command line emoji search.

# Installation

    pip install emo

# Usage

    $ emo pray fire
    🙏 🔥

Emo can also copy straight to the clipboard:

    $ emo tree fire cloud --copy
    🌴 🔥 ☁

There are also --list and --all arguments if you just want to see what's available.

And that's all it does!
